Unit Overview

This unit covers the fundamental primitive data types in programming. Understanding these types is essential for writing efficient code.

Learning Objectives

  • Understand what primitive types are.
  • Identify various primitive types available in programming.
  • Write code using primitive types.
  • Practice and test understanding with exercises and quizzes.

Primitive Types

1. int

The int type represents integer values, which are whole numbers.

int age = 25;

2. double

The double type is used for decimal numbers.

double price = 19.99;

3. boolean

The boolean type can hold two values: true or false.

boolean isStudent = true;

4. char

The char type represents a single character.

char initial = 'A';

Practice Exercises

  1. Declare an int variable and assign a value to it.
  2. Declare a double variable with a decimal value.
  3. Create a boolean variable that indicates if a condition is true.
  4. Declare a char variable for a single letter.
