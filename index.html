AP Computer Science A

Digital Curriculum Hub

AP Computer Science A introduces students to computer science through programming using Java. The course focuses on problem-solving, data structures, algorithms, and object-oriented programming.

Course Overview

This course provides a rigorous introduction to computer science concepts and practices through the Java programming language, preparing students for the AP exam and future academic or professional work in computing.

  • Focus on programming in Java, including syntax, semantics, and standard libraries.
  • Emphasis on object-oriented design — encapsulation, inheritance, and polymorphism.
  • Exploration of the ethical and social implications of computing technologies.
  • Mathematical reasoning forms the foundation for algorithm analysis and problem-solving.
  • No prior computer science experience required — all students are welcome.

Prerequisites

Students should have the following mathematical background before beginning AP Computer Science A.

Completion of first-year algebra
Strong understanding of linear functions
Ability to use Cartesian (x, y) coordinate systems
Mathematical reasoning foundation

Course Units

The course is organized into 10 sequential units that build upon each other.

Unit 1

Primitive Types

Learn the fundamental building blocks of Java: integer, double, boolean, and char data types.

Unit 2

Using Objects

Explore how to create and use objects, call methods, and work with the String class.

Unit 3

Boolean Expressions and if Statements

Understand how programs make decisions using conditional logic and comparison operators.

Unit 4

Iteration

Use while loops, for loops, and do-while loops to repeat blocks of code efficiently.

Unit 5

Writing Classes

Design and implement your own classes with constructors, instance variables, and methods.

Unit 6

Array

Store and manipulate collections of data using fixed-size, single-dimensional arrays.

Unit 7

ArrayList

Work with dynamic lists using the Java ArrayList class and common traversal algorithms.

Unit 8

2D Array

Represent grid-based data structures and practice nested loop traversal patterns.

Unit 9

Inheritance

Apply the is-a relationship between classes, method overriding, and polymorphism in Java.

Unit 10

Recursion

Solve problems by writing methods that call themselves using base and recursive cases.

Big Ideas

Four overarching themes unite the content of the course and connect to real-world computing.

Modularity

Breaking complex problems into smaller, manageable modules — classes, methods, and objects — makes programs easier to build, test, and maintain. Modularity is a cornerstone of professional software engineering.

Variables

Variables store and manipulate data throughout a program's execution. Understanding scope, type, and memory helps programmers write correct and efficient code.

Control

Control structures — conditionals, loops, and method calls — determine how a program flows. Precise control logic is essential for solving well-defined algorithmic problems.

Impact of Computing

Computing technologies affect society in profound ways. Students examine how software decisions carry ethical, legal, and social consequences that extend beyond the classroom.

Computational Thinking Practices

These five practices describe the skills students develop throughout the course.

Program Design and Algorithm Development

Determine required code segments to produce a given output by designing clear, logical algorithms.

Code Logic

Understand what code does, including tracing execution and predicting program behavior.

Code Implementation

Write, modify, and combine code segments to create correct and readable Java programs.

Code Testing

Test and debug programs to identify and correct errors, and verify program correctness.

Documentation

Describe the behavior and purpose of code segments using clear, concise written documentation.

Sample Questions

Examples of the types of questions students can expect on the AP exam.

Multiple-Choice Example

Multiple Choice

Which of the following code segments correctly generates a random integer between 1 and 10 (inclusive) in Java?

  • (int)(Math.random() * 10)
  • (int)(Math.random() * 11)
  • (int)(Math.random() * 10) + 1 ✓
  • (int)(Math.random() * 9) + 1

Free-Response Example

Free Response

Check Digits — Class Design

Students are given a description of a system that uses check digits to validate identification numbers. They are asked to (a) implement a method that computes a check digit for a given number, (b) write a method that verifies whether a full ID is valid, and (c) design a class that encapsulates this behavior with appropriate constructors and instance variables. The question assesses class design, method implementation, and algorithm correctness.
